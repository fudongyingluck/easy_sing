## 数据存储

### 本地存储内容

#### 1. 用户设置 (user_settings)
```javascript
{
  currentMode: 'male',        // 'male' | 'female' | 'custom'
  customModes: [              // 用户自定义的模式列表
    { id: 1, name: '我的高音区', startNote: 'C4', endNote: 'C6' }
  ],
  lastUpdated: '2024-01-15T14:30:00.000Z'
}
```

#### 2. 录音列表 (recordings)
```javascript
[
  {
    id: 'rec_20240115_143022',
    name: '2024-01-15 14:30:22',
    audioFilePath: 'wxfile://.../rec_20240115_143022.mp3',
    pitchDataPath: 'wxfile://.../rec_20240115_143022_pitch.json',
    duration: 155,           // 秒
    fileSize: 1024000,       // 字节
    createTime: '2024-01-15T14:30:22.000Z'
  }
]
```

### 存储限制
- 小程序本地存储上限：10MB
- 单条录音最大：10分钟（约 1-2MB）
- 自动清理阈值：8MB

## 性能优化

### 音频处理优化
- 降低采样率：44100Hz → 22050Hz
- 只分析有效频段：根据当前模式设置 minFreq/maxFreq
- 跳帧处理：每 3 帧分析 1 帧
- 使用 WebAssembly 加速（可选）

### UI 渲染优化
- Canvas 绘制音高曲线
- UI 更新频率限制在 30fps
- 数据精简：只保留最近 100 个数据点
- 虚拟列表：录音列表使用虚拟滚动（可选）

## 微信小程序特有说明

### 录音功能限制
- `wx.saveFileToDisk()` 仅在PC端支持，手机端不支持
- 录音文件保存在小程序私有目录，用户无法在文件App中直接访问
- 无法直接分享录音文件给好友

### 分享功能替代方案
- 提供"保存到手机"引导
- 提示用户手动在文件App中查找并分享
